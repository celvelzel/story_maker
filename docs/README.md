# StoryWeaver 文档中心

本目录包含 StoryWeaver 项目的所有技术文档，按类别组织如下：

## 目录结构

- **api/** - API 接口文档
- **design/** - 设计文档与规范
- **fixes/** - 问题修复报告与解决方案
- **guides/** - 使用指南与部署文档
- **reports/** - 优化与改进报告
- **project/** - 项目规范与计划

## 文档分类

### API 文档 (`api/`)
包含前端接口规范、数据模型和模块 API 说明。

### 设计文档 (`design/`)
包含系统架构设计、算法策略和微调方案等技术设计文档。

### 修复报告 (`fixes/`)
记录各类问题的修复过程、根因分析和解决方案。

### 使用指南 (`guides/`)
提供部署指南、技术路线和数据流说明等实用文档。

### 优化报告 (`reports/`)
包含系统优化、性能改进和功能增强的详细报告。

### 项目文档 (`project/`)
包含项目规范、实现计划和提示词模板。
