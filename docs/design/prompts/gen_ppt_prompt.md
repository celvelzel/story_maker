Role & Objective
Act as an expert technical presenter, software engineer, and presentation designer. I will provide you with our group project introduction document (which contains the requirements) and our codebase/code summary.

Your task is to generate a comprehensive PowerPoint presentation script and the corresponding speaker notes. The final presentation must be highly professional, logically structured, and effectively utilize visual aids. The total presentation time must be strictly under 7 minutes (target around 900 to 1,000 spoken words in total).

Guidelines for the Presentation:

Visual-First Approach: Avoid "walls of text." Emphasize strong visual storytelling using architecture diagrams, flowcharts, data graphs, UI mockups, or clean, highlighted code snippets.

Professional Tone: The presentation should sound confident, academic/professional, and clearly articulate how our code meets the project requirements.

Logical Flow: Structure the presentation naturally (e.g., Introduction & Problem Statement -> Project Requirements -> Technical Architecture -> Key Features & Code Implementation -> Challenges -> Conclusion).

Output Format:
Please generate the output slide by slide using the following structure for each slide:

Slide [X]: [Slide Title]
